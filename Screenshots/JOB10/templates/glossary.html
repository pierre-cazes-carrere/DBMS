Watt (W)

{% if lang == 'en' %} The watt is the unit of power. It indicates how much energy is consumed or produced at a given moment. For example, a 60 W light bulb uses 60 watts when it is on. {% elif lang == 'ar' %} الواط هو وحدة القدرة. يعبّر عن كمية الطاقة المستهلكة أو المنتَجة في لحظة معينة. على سبيل المثال، المصباح بقدرة 60 W يستهلك 60 واط عندما يكون مشغّلاً. {% else %} Le watt est l'unité de puissance. Il indique la quantité d'énergie consommée ou produite à un instant donné. Par exemple, une ampoule de 60 W consomme 60 watts lorsqu'elle est allumée. {% endif %}

Kilowatt (kW)

{% if lang == 'en' %} 1 kW = 1000 W. We use kilowatts for higher power levels, for example for electric heaters or EV charging stations. {% elif lang == 'ar' %} 1 kW = 1000 W. نستخدم الكيلوواط للقدرات الأكبر، مثل قدرة جهاز التدفئة الكهربائي أو محطة شحن السيارات الكهربائية. {% else %} 1 kW = 1000 W. On utilise le kilowatt pour parler de puissances un peu plus grandes, par exemple la puissance d'un chauffage électrique ou d'une borne de recharge de véhicule électrique. {% endif %}

Kilowattheure (kWh)

{% if lang == 'en' %} The kilowatt‑hour is a quantity of energy. This is what your electricity bill measures. 1 kWh is 1 kW used for 1 hour. For example, a 1000 W device running for 1 hour uses 1 kWh. {% elif lang == 'ar' %} الكيلوواط‑ساعة هو كمية من الطاقة. هذا ما تقيسه فاتورة الكهرباء. 1 kWh تعني استخدام قدرة 1 kW لمدة ساعة واحدة. على سبيل المثال، جهاز بقدرة 1000 W يعمل لمدة ساعة واحدة يستهلك 1 kWh. {% else %} Le kilowattheure est une quantité d'énergie. C'est ce que mesure votre facture d'électricité. 1 kWh correspond à une puissance de 1 kW utilisée pendant 1 heure. Par exemple, un appareil de 1000 W utilisé pendant 1 heure consomme 1 kWh. {% endif %}

{% if lang == 'en' %}Carbon intensity (gCO₂/kWh){% elif lang == 'ar' %}كثافة الكربون (gCO₂/kWh){% else %}Intensité carbone (gCO₂/kWh){% endif %}

{% if lang == 'en' %} The amount of CO₂ emitted to produce 1 kWh of electricity. The higher this value, the more carbon‑intensive the electricity is. {% elif lang == 'ar' %} هي كمية CO₂ المنبعثة لإنتاج 1 kWh من الكهرباء. كلما زادت هذه القيمة، كانت الكهرباء أكثر «كربونية». {% else %} C'est la quantité de dioxyde de carbone émise pour produire 1 kWh d'électricité. Plus cette valeur est élevée, plus l'électricité est « carbonée ». {% endif %}

{% if lang == 'en' %}Electricity generation sources{% elif lang == 'ar' %}مصادر إنتاج الكهرباء{% else %}Sources de production d'électricité{% endif %}

{% if lang == 'en' %}Coal (%) {% elif lang == 'ar' %}الفحم (%) {% else %}Charbon (%) {% endif %}

{% if lang == 'en' %} Share of electricity generated from coal‑fired power plants. Coal is one of the most CO₂‑intensive sources per kWh. {% elif lang == 'ar' %} نسبة الكهرباء المولَّدة من محطات تعمل بالفحم. الفحم من أكثر المصادر إصداراً لـ CO₂ لكل kWh منتَج. {% else %} Part de l'électricité produite à partir de centrales au charbon. Le charbon est l'une des sources les plus émettrices de CO₂ par kWh produit. {% endif %}

{% if lang == 'en' %}Gas (%) {% elif lang == 'ar' %}الغاز (%) {% else %}Gaz (%) {% endif %}

{% if lang == 'en' %} Electricity generated from natural gas power plants. Less CO₂‑intensive than coal, but still a fossil fuel. {% elif lang == 'ar' %} الكهرباء المولَّدة من محطات تعمل بالغاز الطبيعي. أقل انبعاثاً من الفحم، لكنها تظلّ طاقة أحفورية. {% else %} Électricité produite à partir de centrales au gaz naturel. Moins émettrice que le charbon, mais reste une énergie fossile. {% endif %}

{% if lang == 'en' %}Oil (%) {% elif lang == 'ar' %}النفط (%) {% else %}Pétrole (%) {% endif %}

{% if lang == 'en' %} Share of electricity generated using petroleum products (fuel oil, diesel, etc.). CO₂ emissions are high, comparable to or higher than coal depending on the use. {% elif lang == 'ar' %} نسبة الكهرباء المولَّدة باستخدام مشتقات النفط (الفيول، الديزل، إلخ). انبعاثات CO₂ مرتفعة، وقد تكون مماثلة أو أعلى من الفحم حسب الاستخدام. {% else %} Part d'électricité produite à partir de produits pétroliers (fioul, diesel, etc.). Les émissions de CO₂ sont élevées, comparables ou supérieures au charbon selon les usages. {% endif %}

{% if lang == 'en' %}Hydro (%) {% elif lang == 'ar' %}كهرومائي (%) {% else %}Hydro (%) {% endif %}

{% if lang == 'en' %} Share of electricity generated from hydropower (dams, run‑of‑river plants). Direct CO₂ emissions per kWh are very low. {% elif lang == 'ar' %} نسبة الكهرباء المولَّدة من الطاقة الكهرومائية (السدود، المحطات النهرية). الانبعاثات المباشرة لـ CO₂ لكل kWh منخفضة جداً. {% else %} Part de l'électricité produite grâce à l'hydroélectricité (barrages, centrales au fil de l'eau). Les émissions directes de CO₂ par kWh sont très faibles. {% endif %}

{% if lang == 'en' %}Renewables (%) {% elif lang == 'ar' %}متجددة (%) {% else %}Renouvelable (%) {% endif %}

{% if lang == 'en' %} Share of renewable energy excluding hydro: mainly solar and wind. Life‑cycle CO₂ emissions are low compared to fossil fuels. {% elif lang == 'ar' %} نسبة الطاقة المتجددة باستثناء الكهرومائية: خاصة الطاقة الشمسية وطاقة الرياح. انبعاثات CO₂ على كامل دورة الحياة منخفضة مقارنة بالطاقات الأحفورية. {% else %} Part des énergies renouvelables hors hydro : principalement solaire et éolien. Les émissions de CO₂ sur le cycle de vie sont faibles par rapport aux énergies fossiles. {% endif %}

{% if lang == 'en' %}Nuclear (%) {% elif lang == 'ar' %}نووي (%) {% else %}Nucléaire (%) {% endif %}

{% if lang == 'en' %} Share of electricity generated from nuclear power plants. Direct CO₂ emissions per kWh are very low, but there are other issues (waste management, safety, etc.). {% elif lang == 'ar' %} نسبة الكهرباء المولَّدة من محطات نووية. الانبعاثات المباشرة لـ CO₂ لكل kWh منخفضة جداً، لكن توجد تحديات أخرى (النفايات، السلامة، إلخ). {% else %} Part de l'électricité produite à partir de centrales nucléaires. Les émissions de CO₂ directes sont très faibles par kWh, mais la technologie pose d'autres enjeux (déchets, sûreté, etc.). {% endif %}

{% if lang == 'en' %}Other useful concepts{% elif lang == 'ar' %}مفاهيم أخرى مفيدة{% else %}Autres notions utiles{% endif %}

{% if lang == 'en' %}Power mix{% elif lang == 'ar' %}المزيج الكهربائي{% else %}Mix électrique{% endif %}

{% if lang == 'en' %} The share of each source used to generate electricity (coal, gas, oil, hydro, renewables, nuclear). The power mix directly affects carbon intensity. {% elif lang == 'ar' %} توزيع مصادر إنتاج الكهرباء (الفحم، الغاز، النفط، الكهرومائي، المتجددة، النووي). يؤثّر المزيج الكهربائي مباشرة في كثافة الكربون. {% else %} Répartition entre les différentes sources de production d'électricité (charbon, gaz, pétrole, hydro, renouvelables, nucléaire). Le mix électrique influence directement l'intensité carbone. {% endif %}

{% if lang == 'en' %}Emission factor{% elif lang == 'ar' %}عامل الانبعاث{% else %}Facteur d'émission{% endif %}

{% if lang == 'en' %} Average amount of CO₂ emitted per kWh for a given source. Example: around 820 gCO₂/kWh for coal, 490 gCO₂/kWh for gas, or ~12 gCO₂/kWh for nuclear. {% elif lang == 'ar' %} كمية CO₂ المتوسطة المنبعثة لكل kWh لمصدر معيّن. مثال: حوالي 820 gCO₂/kWh للفحم، 490 gCO₂/kWh للغاز، أو ~12 gCO₂/kWh للطاقة النووية. {% else %} Quantité moyenne de CO₂ émise par kWh pour une source donnée. Exemple : environ 820 gCO₂/kWh pour le charbon, 490 gCO₂/kWh pour le gaz, ou ~12 gCO₂/kWh pour le nucléaire. {% endif %}

{% if lang == 'en' %}Carbon offsetting{% elif lang == 'ar' %}تعويض الكربون{% else %}Compensation carbone{% endif %}

{% if lang == 'en' %} Actions that aim to “offset” unavoidable CO₂ emissions, for example by planting trees or financing emission‑reduction projects elsewhere. {% elif lang == 'ar' %} إجراءات تهدف إلى «تعويض» الانبعاثات غير القابلة للتقليل، مثل زراعة الأشجار أو تمويل مشاريع خفض الانبعاثات في أماكن أخرى. {% else %} Action qui vise à « compenser » des émissions de CO₂ incompressibles, par exemple via la plantation d'arbres ou des projets de réduction d'émissions ailleurs. {% endif %}
